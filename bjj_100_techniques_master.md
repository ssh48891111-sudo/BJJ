## 📚 기술 라이브러리 (Techniques)

| :--- | :--- | :--- | :--- |
| **Armbar (암바)** | Beginner | 가장 고전적인 팔꿈치 관절 꺾기 기술입니다. | 트라이앵글 초크, 오모플라타 |
| **Triangle Choke (트라이앵글 초크)** | Beginner | 다리로 상대의 목과 팔을 조이는 핵심 기술입니다. | 암바, 오모플라타, 기무라 |
| **Rear Naked Choke (리어 네이키드 초크)** | Beginner | 백 포지션에서 구사하는 가장 강력한 초크입니다. | 바디 트라이앵글 잠금, 보우 앤 애로우 |
| **Kimura (기무라)** | Intermediate | 어깨 관절을 꺾는 강력한 피겨 포 그립 기술입니다. | 기무라 트랩(백테이크), 타리코플라타, 힙 범프 스윗 |

| **Guillotine Choke (길로틴 초크)** | Beginner | 상대의 목을 겨드랑이에 끼워 조이는 기술입니다. | 아나콘다 초크, 다스 초크, 고고플라타 |
| **Omoplata (오모플라타)** | Intermediate | 다리로 상대 팔을 감아 어깨를 꺾습니다. | 트라이앵글 초크, 오모플라타 스윕, 토 홀드 |
| **Bow and Arrow Choke (보우 앤 애로우)** | Intermediate | 라펠을 이용해 활을 쏘듯 조이는 초크입니다. | 크루시픽스 전환, 암바 |
| **D'Arce Choke (다스 초크)** | Advanced | 상대 겨드랑이 밑으로 파고들어 목과 팔을 조입니다. | 아나콘다 초크, 닌자 초크 |
| **Straight Ankle Lock (앵클 락)** | Intermediate | 아킬레스건을 압박하는 기본 하체관절기입니다. | 힐 훅, 아오키 락 |
| **Heel Hook (힐 훅)** | Advanced | 뒤꿈치를 비틀어 무릎 인대를 공격합니다. | 토 홀드, 아웃사이드 힐 훅 |
| **Cross Collar Choke (크로스 칼라 초크)** | Beginner | 양 깃을 교차로 잡아 목을 조릅니다. | 암바, 깃 이용 스윕 |
| **Ezekiel Choke (에제키엘 초크)** | Intermediate | 자신의 소매를 이용해 목을 조릅니다. | 암 트라이앵글, 마운트 컨트롤 |
| **Calf Slicer (카프 슬라이서)** | Advanced | 오금에 정강이를 끼워 종아리를 압박합니다. | 트럭 포지션, 백테이크, 트위스터 |
| **Buggy Choke (버기 초크)** | Advanced | 사이드 아래에서 다리를 걸어 조이는 사파 기술입니다. | 가드 리커버리, 트라이앵글 |
| **Gogoplata (고고플라타)** | Advanced | 정강이로 목을 누르며 조이는 기술입니다. | 오모플라타, 로코플라타 |
| **Estima Lock (에스티마 락)** | Advanced | 상대 발등을 복부에 대고 비틀어 꺾습니다. | 토 홀드, 스윕 연계 |
| **Kneebar (니바)** | Intermediate | 무릎 관절을 반대로 꺾는 하체의 암바입니다. | 토 홀드, 힐 훅 |
| **Toe Hold (토 홀드)** | Intermediate | 발가락 쪽을 잡고 비틀어 발목을 공격합니다. | 니바, 힐 훅 |
| **Twister (트위스터)** | Advanced | 척추를 반대 방향으로 비틀어 꺾는 기술입니다. | 트럭 포지션, 카프 슬라이서 |
| **Anaconda Choke (아나콘다 초크)** | Advanced | 앞 목을 잡고 굴러서 팔과 목을 함께 조입니다. | 다스 초크, 길로틴 |
| **Tarikoplata (타리코플라타)** | Advanced | 다리를 이용해 어깨를 꺾는 기무라 변형입니다. | 기무라, 암바 |
| **Wrist Lock (손목 꺾기)** | Intermediate | 상대의 손목을 갑작스럽게 꺾어 제압합니다. | 암바, 기무라, 트라이앵글 |
| **Paper Cutter Choke (페이퍼 커터)** | Intermediate | 사이드에서 팔꿈치를 교차해 라펠로 조입니다. | 노스사우스 초크, 암바 |
| **North South Choke (노스 사우스 초크)** | Advanced | 남북 포지션에서 어깨로 상대 목을 조릅니다. | 길로틴, 기무라 |
| **Von Flue Choke (본 플루 초크)** | Intermediate | 상대가 길로틴을 잡고 있을 때 어깨로 압박해 역습합니다. | 사이드 컨트롤 유지 |
| **Peruvian Necktie (페루비안 넥타이)** | Advanced | 프론트 헤드락에서 다리를 머리 위로 올려 조입니다. | 아나콘다, 다스 초크 |
| **Japanese Necktie (재패니즈 넥타이)** | Advanced | 다스 초크 시도가 막혔을 때 다리를 걸어 조입니다. | 다스 초크 |
| **Baseball Bat Choke (베이스볼 배트)** | Intermediate | 야구 방망이를 잡듯 양 깃을 잡고 조입니다. | 니온밸리, 가드 패스 허용 후 역습 |
| **Belly Down Armbar (벨리 다운 암바)** | Intermediate | 배를 바닥으로 향하게 하여 체중을 실어 암바를 겁니다. | 암바, 트라이앵글 |

| 점수 | 포지션/액션 | 설명 |

| **4 PTS** | **Back Control (백 컨트롤)** | 상대의 등 뒤에서 두 다리로 훅을 걸어 완전히 제압한 상태. |

| **2 PTS** | **Knee on Belly (니온밸리)** | 상대의 배 위에 무릎을 올리고 반대쪽 다리는 펴서 중심을 잡은 상태. |
